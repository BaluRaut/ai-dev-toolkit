⚡ Usage

One command to generate entire features

Terminal — zsh
# Full workflow: Jira + Figma + API → Code
python main.py --jira PROJ-123 --figma "https://figma.com/file/..." --swagger "https://api.com/swagger.json"

# Interactive mode — paste context step by step
python main.py --interactive

# Code review
python main.py --review src/components/Profile.tsx

# Generate tests
python main.py --test src/services/userService.ts

# Fix a bug
python main.py --fix "TypeError: Cannot read property 'map' of undefined"
